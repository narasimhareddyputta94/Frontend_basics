Overflow: Visible

This text is so long that it definitely overflows the container. With overflow set to visible, all of the overflow content will be visible outside the container.

Overflow: Hidden

This text is so long that it definitely overflows the container. With overflow set to hidden, the extra content that does not fit within the container is clipped, and no scrollbars are provided.

Overflow: Scroll

This text is so long that it definitely overflows the container. With overflow set to scroll, scrollbars are always visible, and you can scroll to see the hidden content.

Overflow: Auto

This text is so long that it definitely overflows the container. With overflow set to auto, scrollbars will appear automatically only if they are necessary to see the content that overflows.
